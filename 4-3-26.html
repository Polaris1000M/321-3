    Section 3 4-3-26

    Generated \(\sigma\)-Algebras, Borel Sets, Carathéodory Criterion.
    🔗

    Example 3.1.

    Let \(\mathcal F \subseteq \mathcal P(\R)\) such that
    \begin{equation*} \mathcal F = \{A \subseteq \R \mid \textrm{either } A \textrm{ open or } A \textrm{ closed}\}\text{.} \end{equation*}
    🔗
    Since \(\mathcal F\) is not closed under countable unions (or even finite unions) \(\mathcal F\) is not a \(\sigma\)-algebra.
    🔗
    🔗

    Definition 3.2. \(\sigma\)-Algebra Generated by \(\mathcal F\).

    Let \(\mathcal F \subseteq \mathcal P(X)\) be a collection of subsets of \(X\text{.}\) Let \(\sigma(\mathcal F)\) be the smallest \(\sigma\)-algebra on \(X\) containing \(\mathcal F\text{.}\)
    🔗
    Call \(\sigma(\mathcal F)\) the \(\sigma\)-algebra generated by \(\mathcal F\text{.}\)
    🔗
    🔗

    Definition 3.3. Borel \(\sigma\)-Algebra.

    Let \(\mathcal B = \sigma(\{\textrm{open sets}\}) = \sigma(\{\textrm{closed sets}\})\text{.}\) Then, \(\mathcal B\) is called the Borel \(\sigma\)-algebra.
    🔗
    The elements of \(\mathcal B\) are called Borel sets.
    🔗
    🔗

    Example 3.4.

    Let \(\mathcal F = \{[1, 2 - \frac{1}{n}] \mid n \in \N\}\) be a collection of closed sets. \(\bigcup_{n = 1}^\infty [1, 2 - \frac{1}{n}] = [1, 2)\text{.}\) Consequently, \(\mathcal F\) is not closed under countable unions and is not a \(\sigma\)-algebra.
    🔗
    🔗

    Example 3.5.

    Let \(X = \{1, 2, 3, 4, 5\}\text{.}\) Let \(\mathcal F = \{\{1, 3\}\}\text{.}\) Then,
    \begin{equation*} \sigma(\mathcal F) = \{\emptyset, X, \{1, 3\}, \{2, 4, 5\}\}\text{.} \end{equation*}
    🔗
    🔗

    Definition 3.6. Borel \(\sigma\)-Algebra of Lebesgue Measurable Sets.

    Let \(\mathcal M\) be the \(\sigma\)-algebra on \(\R\) generated by Borel sets and null sets.
    🔗
    Later we will see that \(\mathcal M\) is the \(\sigma\)-algebra of Lebesgue measurable sets.
    🔗
    We sometimes call \(\mathcal M\) the completion of \(\mathcal B\text{.}\)
    🔗
    🔗

    Definition 3.7. Carathéodory Criterion.

    We say \(A \subseteq \R\) is in \(\mathcal M_0\) if for any subset \(X \subseteq \R\)
    \begin{equation*} \mu^*(X) = \mu^*(X \cap A) + \mu^*(X \cap A^{\mathsf c})\text{.} \end{equation*}
    🔗
    \(\mathcal M_0\) is equivalent to \(\mathcal M\) and is also the \(\sigma\)-algebra of Lebesgue measurable sets.
    🔗
    🔗

    Remark 3.8.

    We always have \(\mu^*(X) \le \mu^*(X \cap A) + \mu^*(X \cap A^{\mathsf c})\) by countable subadditivity. Consequently, to show that \(A \in \mathcal M_0\text{,}\) it suffices to prove \(\mu^*(X) \ge \mu^*(X \cap A) + \mu^*(X \cap A^{\mathsf c})\text{.}\)
    🔗
    We will also see that Borel sets and null sets satisfy the Carathéodory Criterion.
    🔗
    🔗

    Proposition 3.9. Measurability of Null Sets.

    Null sets are measurable.
    🔗
    🔗

    Proof.

    Let \(A \subset \R\) be a null set. Take any \(A \subseteq \R\text{.}\) Then,
    \begin{equation*} \mu^*(X \cap A) + \mu^*(X \cap A^{\mathsf c}) \underset{\textrm{M.}}{\le} + \mu^*(A) + \mu^*(X \cap A^{\mathsf c}) \underset{\textrm{M.}}{\le} 0 + \mu^*(X)\text{.} \end{equation*}
    🔗
    🔗

    Observation 3.10. \(\mathcal M_0\) Satifies First Two Properties of \(\sigma\)-Algebras.

    \(A \in \mathcal M_0\) if and only if \(A^{\mathsf c} \in \mathcal M_0\text{.}\)
    🔗
    \(\emptyset \in \mathcal M_0\) so \(\R \in \mathcal M_0\text{.}\)
    🔗
    Thus, \(\mathcal M_0\) satisfies the first two properties of \(\sigma\)-algebras outlined in Definition 2.8. It remains to show that \(\mathcal M_0\) is closed under countable unions.
    🔗
    🔗

    Proposition 3.11. Finite Unions of \(\mathcal M_0\).

    Let \(A, B \in \mathcal M_0\text{.}\) Then, \(A \cup B \in \mathcal M_0\text{.}\)
    🔗
    🔗

    Proof.

    Note that
    \begin{equation*} X \cap (A \cup B) = (X \cap B) \cup (X \cap A \cap B^{\mathsf c})\text{.} \end{equation*}
    Additionally,
    \begin{align} \mu^*(X \cap B^{\mathsf c}) \amp = \mu^*((X \cap B^{\mathsf c}) \cap A) + \mu^*((X \cap B^{\mathsf c}) \cap A^{\mathsf c})\notag\\ \amp = \mu^*(X \cap B^{\mathsf c} \cap A) + \mu^*(X \cap (A \cup B)^{\mathsf c})\tag{3.1} \end{align}
    where the first equality holds from \(A \in \mathcal M_0\) and the second from \(B^{\mathsf c} \cap A^{\mathsf c} = (A \cup B)^{\mathsf c}\text{.}\)
    🔗
    Consequently,
    \begin{align*} \mu^*(X \cap (A \cup B)) + \mu^*(X \cap (A \cup B)^{\mathsf c}) \amp \underset{\textrm{S.A.}}{\le} \mu^*(X \cap B) + \mu^*(X \cap A \cap B^{\mathsf c})\\ \amp \phantom{\underset{\textrm{S.A.}}{\le}} + \mu^*(X \cap (A \cup B)^{\mathsf c})\\ \amp= \mu^*(A \cap B) + \mu^*(X \cap B^{\mathsf c})\\ \amp = \mu^*(X)\text{.} \end{align*}
    Thus, \(\mu^*(X \cap (A \cup B)) + \mu^*(X \cap (A \cup B)^{\mathsf c}) \le \mu^*(X)\text{,}\) as required.
    🔗
    🔗

    Proposition 3.12. Disjoint Elements of \(\mathcal M_0\).

    Let \(A, B \in \mathcal M_0\) be disjoint. Then, for any \(X \subseteq \R\text{,}\)
    \begin{equation*} \mu^*(X \cap (A \cup B)) = \mu^*(X \cap A) + \mu^*(X \cap B)\text{.} \end{equation*}
    🔗
    🔗

    Proof.

    \begin{align*} \mu^*(X \cap (A \cup B)) \amp = \mu^*(X \cap (A \cup B) \cap A) + \mu^*(X \cap (A \cup B) \cap A^{\mathsf c})\\ \amp = \mu^*(X \cap A) + \mu^*(X \cap B)\text{.} \end{align*}
    🔗

    Corollary 3.13. Finite Additivity of Disjoint Elements of \(\mathcal M_0\).

    Let \(X = \R\text{.}\) If \(A, B \in \mathcal M_0\text{,}\) \(A \cap B = \emptyset\text{,}\) we have \(\mu^*(A \cup B) = \mu^*(A) + \mu^*(B)\) from Proposition 3.12 and Definition 3.7. Thus, for measurable sets \(A, B \in \mathcal M_0\) that are disjoint, we have finite additivity.
    🔗
    🔗

    Proposition 3.14. Closedness of \(\mathcal M_0\) Under Countable Unions.

    Let \(\{A_i\}_{i = 1}^\infty \subseteq \mathcal M_0\text{.}\) Then, \(\bigcup_{i = 1}^\infty A_i \in \mathcal M_0\text{.}\)
    🔗
    🔗

    Proof.

    Let \(E_k = \bigcup_{j = 1}^k A_j\text{,}\) let \(B_k = A_k \backslash E_{k - 1} = A_k \cap (E_{k - 1})^{\mathsf c}\) for \(k > 1\text{,}\) and assume \(A_1 = E_1 = B_1\text{.}\)
    🔗
    Firstly, \(E_\infty = \bigcup_{i = 1}^\infty A_i\text{.}\) Then, \(E_\infty = \bigcup_{i = 1}^\infty B_i\) as well. Finally, \(B_i \cap B_j = \emptyset\) if \(i \ne j\text{,}\) so the set of \(B_i\) is pairwise disjoint.
    🔗
    Given \(X \subseteq \R\) and \(N \in \N\text{,}\)
    \begin{align*} \mu^*(X) \amp = \mu^*(X \cap B_N) + \mu^*(X \cap E_N^{\mathsf c})\\ \amp \underset{\textrm{M.}}{\ge} \mu^*(X \cap E_N) + \mu^*(X \cap E_\infty^{\mathsf c})\\ \amp = \sum_{i = 1}^{N} \mu^*(X \cap B_i) + \mu^*(X \cap E_\infty^{\mathsf c}) \end{align*}
    where the final inequality holds from .
    🔗
    Since this holds for all \(N \in \N\text{,}\) it follows that
    \begin{align*} \mu^*(X) \amp \ge \sum_{i = 1}^{\infty} \mu^*(X \cap B_i) + \mu^*(X \cap E_\infty^{\mathsf c})\\ \amp \underset{\textrm{S.A.}}{\ge} \mu^*\left(\bigcup_{i = 1}^\infty (X \cap B_i)\right) + \mu^*(X \cap E_\infty^{\mathsf c})\\ \amp = \mu^*(X \cap E_\infty) + \mu^*(X \cap E_\infty^{\mathsf c}) \end{align*}
    so \(E_\infty\) is measurable.
    🔗
    As \(E_\infty = bigcup_{i = 1}^\infty A\text{,}\) it follows that \(\bigcup_{i = 1}^\infty A \in \mathcal M_0\text{,}\) as required.
    🔗
    🔗

    Corollary 3.15. \(\mathcal M_0\) is a \(\sigma\)-Algebra.

    From Observation 3.10 and Proposition 3.14, \(\mathcal M_0\) is a \(\sigma\)-algebra.

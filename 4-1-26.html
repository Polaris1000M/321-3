    Section 2 4-1-26

    Properties of the Outer Measure, Introduction to \(\sigma\)-Algebras.
    🔗

    Proposition 2.1. Normalization of Outer Measure on Compact Intervals.

    Let \(I = [a, b]\) be a compact interval. Then, \(\mu^*(I) = b - a\text{.}\)
    🔗
    🔗

    Proof.

    Firstly, \(\mu^*(I) \le b - a\text{.}\) \(I \subset (a - \varepsilon, b + \varepsilon)\) so \(\{(a - \varepsilon, b + \varepsilon)\} \in \mathcal U(I)\text{.}\) Thus,
    \begin{equation*} \mu^*(I) \le \inf_{\varepsilon > 0} \{\textrm{len}(a - \varepsilon, b + \varepsilon)\} = \inf_{\varepsilon > 0} \{b - a + 2\varepsilon\} = b - a. \end{equation*}
    🔗
    We now show that \(\mu^*(I) \ge b - a\text{.}\) Let \(\mathcal U' \in \mathcal U(I)\) be a cover of \(I\) by countably many open intervals. By compactness, there is a finite \(\mathcal U \subseteq \mathcal U'\) such that \(I \subset \bigcup_{U \in \mathcal U} U\text{.}\)
    🔗
    Since \(\mathcal U\) is a cover of \(I\text{,}\) there exists \(J_1 = (a_1, b_1) \in \mathcal U\) with \(a \in J_1\text{.}\) If \(b_1 \ge b\text{,}\) then
    \begin{equation*} \sum_{U \in \mathcal U} \textrm{len } U \ge \textrm{len } J_1 = b_1 - a_1 \ge b - a. \end{equation*}
    Otherwise, \(b_1 < b\) and since \(a \in J_1\text{,}\) we have \(b_1 \in I\text{.}\) Then, pick \(J_2 = (a_2, b_2)\) such that \(b_1 \in J_2\text{.}\) Again, either \(b_2 \ge b\) or \(b_2 \in I\text{.}\)
    🔗
    Suppose we have \(J_1 = (a_1, b_2), J_2 = (a_2, b_2), \ldots, J_{n - 1} = (a_{n - 1}, b_{n - 1})\) such that
    1. \(a \in J_1\text{.}\)
      🔗
    2. \(b_i \in J_{i + 1}\) for \(1 \le i \le n - 2\text{,}\) i.e., \(a_{i + 1} < b_i < b_{i + 1}\text{.}\)
      🔗
    🔗
    If \(b_{n - 1} \le b\text{,}\) then there is \(J_n \in \mathcal U\text{,}\) \(J_n = (a_n, b_n)\) and \(b_{n - 1} \in J_n\text{.}\) Since \(\mathcal U\) is finite, this process must terminate and we can assume that \(b_n > b\text{.}\) Then
    \begin{align*} \sum_{U \in \mathcal U} \textrm{len } U &\ge \sum_{i = 1}^{n} \textrm{len } J_i\\ & = \sum_{i = 1}^{n} (b_i - a_i)\\ & = b_n + \sum_{i = 1}^{n - 1} (b_i - a_{i + 1}) - a_1\\ & \ge b_n - a_1\\ & \ge b - a. \end{align*}
    🔗
    Since \(\mathcal U \subseteq \mathcal U'\text{,}\) it follows that \(\sum_{U \in \mathcal U} \textrm{len } U \le \sum_{U \in \mathcal U'} \textrm{len } U\text{.}\) Consequently, for every \(\mathcal U' \in \mathcal U(I)\text{,}\) \(\sum_{U \in \mathcal U'} \textrm{len } U \ge b - a\text{.}\) As \(\mu^*(I) = \inf_{\mathcal U' \in \mathcal U(I)} \sum_{U \in \mathcal U'} \textrm{len } U\text{,}\) it follows that \(\mu^*(I) \ge b - a\text{.}\)
    🔗
    We have shown that \(\mu^*(I) \le b - a\) and \(\mu^*(I) \ge b - a\text{.}\) Therefore, \(\mu^*(I) = b - a\text{,}\) as required.
    🔗

    Proposition 2.2. Normalization of Outer Measure on Bounded Intervals.

    Let \(I \subset \R\) be a bounded interval with endpoints \(a < b\text{.}\) Then, \(\mu^*(I) = b - a\text{.}\)
    🔗
    🔗

    Proof.

    \([a + \varepsilon, b - \varepsilon] \subset I \subset [a, b]\text{.}\) By monotonicity, \(b - a - 2\varepsilon \le \mu^*(I) \le b - a\text{.}\) It follows that \(\mu^*(I) = b - a\text{.}\)
    🔗
    🔗

    Proposition 2.3. Outer Measure on Unbounded Intervals.

    If \(I\) is an unbounded interval, then \(\mu^*(I) = \infty\text{.}\)
    🔗
    🔗

    Proof.

    Without loss of generality, let \(I = (a, \infty)\text{.}\) \(\left[a + \varepsilon, \frac{1}{\varepsilon}\right] \subset I\text{.}\) As \(\varepsilon \rightarrow 0\) we have \(\mu^*(I) \rightarrow \infty\text{,}\) as required.
    🔗
    🔗

    Proposition 2.4. Translation Invariance.

    Let \(E \subset \R\) and for \(t \in \R\) write \(E + t = \{x + t \mid x \in E\}\text{.}\) Then, \(\mu^*(E) = \mu^*(E + t)\text{.}\)
    🔗
    🔗

    Proof.

    It is true that \(\{U_i\} \in \mathcal U(E)\) if and only if \(\{U_i + t\} \in \mathcal U(E + t)\text{.}\) Since \(\textrm{len } U_i = \textrm{len} (U_i + t)\text{,}\) it follows that \(\sum U_i = \sum \textrm{len} (U_i + t)\) and \(\mu^*(E) = \mu^*(E + t)\text{.}\)
    🔗
    🔗

    Proposition 2.5. Subadditivity of Outer Measure.

    Let \(A_1, A_2, \ldots \subset \R\text{.}\) Then, \(\mu^*\left(\bigcup_{i = 1}^\infty A_i\right) \le \sum_{i = 1}^{\infty} \mu^*(A_i)\text{.}\)
    🔗
    🔗

    Proof.

    For each \(i \in \N\text{,}\) pick \(\{U_j\}_i \in \mathcal U(A_i)\) with \(\sum \textrm{len } U_j \le \mu^*(A_i) + \frac{\varepsilon}{2^i}\text{.}\) Such a \(\{U_j\}_i\) exists since \(\mu^*(A_i)\) is an infimum.
    🔗
    Since \(\bigcup_{U_j \in \{U_j\}_i} U_j\) is a cover of \(A_i\text{,}\) it follows that \(\bigcup_{i = 1}^\infty \{U_j\}_i\) is a cover of \(\bigcup A_i\) by open intervals. Therefore,
    \begin{equation*} \sum_{i = 1}^\infty \sum_{U_j \in \{U_j\}_i} \textrm{len } U_j \le \sum_{i = 1}^{\infty} \left(\mu^*(A_i) + \frac{\varepsilon}{2^i}\right) = \sum_{i = 1}^\infty \mu^*(A_i) + \varepsilon. \end{equation*}
    As \(\mu^*\left(\bigcup_{i = 1}^\infty A_i\right) \le \sum_{i = 1}^\infty \sum_{U_j \in \{U_j\}_i} \textrm{len } U_j\text{,}\) it follows that \(\mu^*\left(\bigcup_{i = 1}^\infty A_i\right) \le \sum_{i = 1}^{\infty} \mu^*(A_i)\text{,}\) as required.
    🔗
    🔗

    Remark 2.6. Sets Without Outer Measure.

    There exists \(B \subset [0, 1]\) such that \(\mu^*(B) + \mu^*([0, 1] \backslash B) > 1\text{.}\) Non-measurable sets exist and \(\mu^*\) might fail to be finitely additive on disjoint sets.
    🔗
    🔗

    Proposition 2.7. Outer Regularity of Outer Measure.

    Let \(A \subset \R\) with \(\mu^*(A) < \infty\text{.}\) Then, there is an open set \(U\) with \(A \subset U\) and \(\mu^*(A) \le \mu^*(U) \le \mu^*(A) + \varepsilon\text{.}\)
    🔗
    🔗

    Proof.

    Let \(\varepsilon > 0\text{.}\) Take some \(\{U_j\} \in \mathcal U(A)\) with \(\sum \textrm{len } U_j \le \mu^*(A) + \varepsilon\text{.}\) Set \(U = \bigcup U_j\text{.}\) Then, \(A \subseteq U\) and
    \begin{equation*} \mu^*(U) \le \sum \textrm{len } U_j \le \mu^*(A) + \varepsilon. \end{equation*}
    🔗
    🔗

    Definition 2.8. \(\sigma\)-Algebras.

    A subset \(\mathcal A \subseteq \mathcal P(X)\) is a \(\sigma\)-algebra if
    1. \(X \in \mathcal A\text{.}\)
      🔗
    2. \(\mathcal A\) is closed under complements, i.e., if \(E \in \mathcal A\text{,}\) then \(E^\mathsf{c} = X \backslash E \in \mathcal A\text{.}\)
      🔗
    3. \(\mathcal A\) is closed under countable unions. If \(\{A_i\}_{i = 1}^\infty \subseteq A\text{,}\) then \(\bigcup_{i = 1}^\infty A_i \in \mathcal A\text{.}\)
      🔗
    🔗
    Note that \(\sigma\) means "countable" and \(\mathcal P(X)\) denotes the power set of \(X\text{.}\)
    🔗
    🔗

    Remark 2.9. Intersections and \(\sigma\)-Algebras.

    If \(\mathcal A\) is a \(\sigma\)-algebra on \(X\text{,}\) then for any \(\{A_i\}_{i = 1}^\infty \subseteq \mathcal A\text{,}\) \(\bigcap_{i = 1}^\infty A_i \in \mathcal A\text{.}\) This follows from De Morgan’s Laws as \(\bigcap_{i = 1}^\infty A_i = \left(\bigcup_{i = 1}^\infty A_i^\mathsf{c}\right)^\mathsf{c}\text{.}\)

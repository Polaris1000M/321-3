    Section 1 3-31-26

    Example 1.1. Inconsistencies of Riemann Integration.

    Assume \(f: [0, 3] \rightarrow \R\) and
    \begin{equation*} f(x) = \begin{cases} 3 \quad & x \in [0, 1] \backslash \Q\\ 7 \quad & x \in [1, 3] \backslash \Q\\ 1 \quad & x \in [0, 3] \cap \Q \text{.}\end{cases} \end{equation*}
    \(\int_{0}^{3} f(x)dx\) is undefined since \(f\) is discontinuous at every point.
    🔗
    What is the "average" value of \(f\text{?}\)
    \begin{equation*} \frac{1}{3} \cdot \int_{0}^{3} f(x)dx = \frac{1}{3} (3 \cdot 1 + 7 \cdot 2 + 1 \cdot 0) = \frac{17}{3}\text{.} \end{equation*}
    🔗
    Let \(\Q \cap [0, 3] = \{q_1, q_2, \cdots\}\text{,}\) let \(Q_n = \{q_1, \ldots, q_n\}\text{,}\) and let
    \begin{equation*} f_n(x) = \begin{cases} 3 \quad & x \in [0, 1] \backslash \Q_n\\ 7 \quad & x \in [1, 3] \backslash \Q_n\\ 1 \quad & x \in Q_n \text{.}\end{cases} \end{equation*}
    Then, \(\int_{0}^{3} f_n(x)dx\) exists and is equal to 17. Furthermore, \(f_n \rightarrow f\) pointwise in a "controlled" way (since \(f\) and \(f_n\) are bounded).
    🔗
    🔗

    Claim 1.2. Motivation for Lebesgue Theory.

    Riemann integration is "bad." It does not hold in the limit and it does not respect the idea of an "average" value. This motivates Lebesgue theory.
    🔗
    🔗

    Definition 1.3. Measure.

    Let \(\mu: \{\textrm{subsets of } \R\} \rightarrow \R\) be a measure of subsets of \(\R\text{.}\) Then, \(\mu\) satisfies
    1. Nonnegativity. \(\mu(E) \in [0, \infty)\text{.}\)
      🔗
    2. Monotonicity. If \(A \subseteq B\text{,}\) \(\mu(A) \le \mu(B)\text{.}\)
      🔗
    3. Normalization. \(\mu([0, 1]) = 1\text{.}\)
      🔗
    4. Translation invariance. Assume \(A + t = \{x + t \mid x \in A\}\text{.}\) Then, \(\mu(A + t) = \mu(A)\text{.}\)
      🔗
    5. Countable additivity. Let \(A_1, A_2, \ldots \subseteq \R\) be pairwise disjoint. Then, \(\mu \left(\bigcup_{i = 1}^\infty\right)= \sum_{i = 1}^{\infty} \mu(A_i)\)
      🔗
    🔗
    🔗

    Remark 1.4. Nonexistence of a Measure.

    We want a measure \(\mu\) to define a reasonable notion of the "size" of subsets \(E \subseteq \R\text{.}\) However, no such \(\mu\) exists! This is mainly due to the fifth property. The solution is to restrict the domain of \(\mu\) from certain subsets of \(\R\text{.}\)
    🔗
    🔗

    Definition 1.5. Lebesgue Outer Measure.

    Let \(U = (a, b)\) be an open interval. Define \(\textrm{len } U = b - a\text{.}\) Given \(E \subseteq \R\text{,}\) let \(\mathcal U = \{U_i\}\) be a cover of \(E\) by open intervals.
    🔗
    By the Second Axiom of Countability, we can cover \(E\) by countably many intervals. Let \(\mathcal U(E) = \{\textrm{covers of } E \textrm{ by countably many open intervals}\}\text{.}\)
    🔗
    Then,
    \begin{equation*} \mu^*(E) = \inf \left\{\sum_{\{U_n\} \in \mathcal U(E)} \textrm{len}(U_n)\right\}. \end{equation*}
    🔗
    Note that \(\mu^*\) is not a measure.
    🔗
    🔗

    Example 1.6. Outer Measure Examples.

    Let \(A = (a, b)\text{.}\) Then, \(\mu^*(A) = b - a\text{.}\) Since \(A \subseteq (a, b)\text{,}\) we have \(\mu^*(A) \le b - a\text{.}\)
    🔗
    Let \(A = \{7\}\) be a singleton. Then, \(A \subseteq (7 - \varepsilon, 7 + \varepsilon)\) so \(\mu^*(A) \le 2\varepsilon\text{.}\) Since \(\varepsilon\) is arbitrary, \(\mu^*(A) = 0\text{.}\)
    🔗
    Assume \(A = \Q\text{.}\) Let \(\Q = \{q_1, q_2, \ldots\}\text{.}\) For \(n \in \N\text{,}\) let \(U_n = \left(q_n - \frac{\varepsilon}{2^n}, q_n + \frac{\varepsilon}{2^n}\right)\text{.}\) Clearly, \(\Q \subseteq \bigcup_{n = 1}^\infty U_n\text{,}\) so \(\{U_n\}_{n = 1}^\infty\) covers \(\Q\) by countably many open intervals. Then,
    \begin{equation*} \sum_{n = 1}^{\infty} \textrm{len } U_n = \sum_{n = 1}^{\infty} \frac{\varepsilon}{2^{n - 1}} = \varepsilon \sum_{k = 0}^{\infty} \frac{1}{2^k} = 2\varepsilon. \end{equation*}
    Thus, we have \(\mu^*(A) = \mu^*(\Q) = 0\text{.}\)
    🔗
    🔗

    Corollary 1.7.

    Countable sets have outer measure \(0\text{.}\)
    🔗
    🔗

    Definition 1.8. Null Set.

    \(E \subseteq \R\) is a null set if \(\mu^*(E) = 0\text{.}\)
    🔗
    Null sets are always measurable.
    🔗
    🔗

    Example 1.9. Middle \(\frac{1}{3}\) Cantor set.

    The middle \(\frac{1}{3}\) Cantor set is a null set but it is uncountable.
    🔗
    🔗

    Proposition 1.10. Monotonicity of the Outer Measure.

    Given \(A \subseteq B\text{,}\) we have \(\mu^*(A) \le \mu^*(B)\text{.}\)
    🔗
    🔗

    Proof.

    Suppose \(A \subseteq B\text{.}\) Then, let \(\{U_n\} \in \mathcal U(B)\) be a cover of \(B\) by countably many open intervals. It follows that \(\{U_n\}\) is also a cover of \(A\text{.}\) Since \(\mathcal U(B) \subseteq \mathcal U(A)\)
    \begin{equation*} \inf \sum_{\{U_n\} \in \mathcal U(A)} \textrm{len } U_n \le \inf \sum_{\{U_n\} \in \mathcal U(B)} \textrm{len } U_n \end{equation*}
    by the properties of the infimum.

    Section 5 4-8-26

    Theorem 5.1. The measure on \(\mathcal M\) exists.

    There exists a unique set function \(\mu: \mathcal M \rightarrow [-\infty, \infty]\) satisfying
    1. Nonnegativity.
      🔗
    2. Monotonicity.
      🔗
    3. Normalization.
      🔗
    4. Translation invariance.
      🔗
    5. Countable additivity.
      🔗
    6. Outer regularity. Given \(A \in \mathcal M\) and \(\varepsilon > 0\text{,}\) there is \(\mathcal U\) open with \(A \subseteq \mathcal U\) and \(\mu(\mathcal U \backslash A) < \varepsilon\text{.}\)
      🔗
    🔗
    🔗

    Remark 5.2.

    Given \(A \in \mathcal M\text{,}\) \(\mu(A) = \mu^*(A)\text{.}\) The Legesgue measure is just the restriction of the outer measure \(\mu^*\) to \(\mathcal M\text{.}\)
    🔗
    🔗

    Proposition 5.3. Inner regularity.

    Given \(A \in \mathcal M\) and \(\varepsilon > 0\text{,}\) there is a closed \(F \subseteq \R\) with \(F \subseteq A\) and \(\mu(A\backslash F) < \varepsilon\text{.}\)
    🔗
    🔗

    Proof.

    Let \(B = A^{\mathsf c}\text{.}\) By outer regularity, there is open \(\mathcal U\) with \(B \subseteq \mathcal U\) and \(\mu(\mathcal U\backslash B) < \varepsilon\text{.}\) Let \(F = \mathcal U^{\mathsf c}\text{.}\) Then since \(B \subseteq \mathcal U\text{,}\) \(A = B^{\mathsf c} \supseteq \mathcal U^{\mathsf c} = F\) and \(\mathcal U \backslash B = \mathcal U \cap B^{\mathsf c} = F^{\mathsf c} \cap A = A \backslash F\text{,}\) so \(\mu(A \backslash F) = \mu(\mathcal U \backslash B) < \varepsilon\text{.}\)
    🔗
    🔗

    Proposition 5.4. Continuity of measure.

    1. Given an increasing sequence \(A_1 \subseteq A_2 \subseteq A_3 \subseteq \cdots \in \mathcal M\text{,}\) \(\mu\left(\bigcap_{i = 1}^\infty B_i\right) = \lim_{i \rightarrow \infty} \mu(B_i)\text{.}\)
      🔗
    2. Given a descending sequence \(B_1 \supseteq B_2 \supseteq \cdots \in \mathcal M\) with \(\mu(B_1) < \varepsilon\text{,}\) \(\mu\left(\bigcap_{i = 1}^\infty B_i\right) = \lim_{i \rightarrow \infty} \mu(B_i)\text{.}\)
      🔗
    🔗

    Proof.

    1. Let \(G_1 = A_1\) and let \(G_k = A_k \backslash A_{k - 1}\text{.}\) Then, \(A_k = \bigcup_{i = 1}^k G_i\) and \(G_i \cap G_j = \emptyset\) for \(i \ne j\text{.}\) Then,
      \begin{align*} \mu\left(\bigcup_{j = 1}^\infty A_j\right) \amp= \mu\left(\bigcup_{j = 1}^\infty G_j\right) = \sum_{j = 1}^{\infty} \mu(G_j)\\ \amp= \lim_{N \rightarrow \infty} \sum_{j = 1}^{N} \mu(G_j) = \lim_{N \rightarrow \infty} \mu(A_N)\text{.} \end{align*}
      🔗
    2. First note that \(\{\mu(B_n)\}\) is non-increasing and bounded below by \(0\) so \(\lim_{n \rightarrow \infty} \mu(B_n)\) exists.
      🔗
      Let \(A_i = B_1 \backslash B_i\text{.}\) Then, \(A_1 \subseteq A_2 \subseteq A_3 \subseteq \cdots\text{.}\) Also,
      \begin{align*} B_1 \backslash \bigcap_{j = 1}^\infty B_j \amp= B_1 \cap \left(\bigcap_{j = 1}^\infty B_j\right)^{\mathsf c} = B_1 \cap \bigcup_{j = 1}^\infty B_j^{\mathsf c}\\ \amp= \bigcup_{j = 1}^\infty\left(B_1 \cap B_j^{\mathsf c}\right) = \bigcup_{j = 1}^\infty A_j\text{.} \end{align*}
      🔗
      It follows that \(B_1 = \left(B_1 \backslash \bigcap_{j = 1}^\infty B_j\right) \sqcup \left(\bigcap_{j = 1}^\infty B_j\right)\) so \(\mu(B_1) = \mu\left(B_1 \backslash \bigcap_{j = 1}^\infty B_j\right) + \mu\left(\bigcap_{j = 1}^\infty B_j\right)\) due to disjointness and measurability. Thus,
      \begin{align*} \mu\left(\bigcap_{j = 1}^\infty B_j\right) \amp= \mu(B_1) - \mu\left(B_1 \backslash \bigcap_{j = 1}^\infty B_j\right)\\ \amp= \mu(B_1) - \lim_{n \rightarrow \infty} \mu(A_n)\\ \amp= \mu(B_1) - \lim_{n \rightarrow \infty} (\mu(B_1) - \mu(B_n))\\ \amp= \lim_{n \rightarrow \infty} \mu(B_n) \end{align*}
      since \(A_n = B_1 \backslash B_n\text{,}\) \(B_1 = B_1 \cap B_n^{\mathsf c} \cup B_1 \cap B_n = B_1 \backslash B_n \cup B_n\text{,}\) and \(B_1\) is finite.
      🔗
      🔗
    🔗

    Proposition 5.5.

    For \(A, B \in \mathcal M\text{,}\) \(\mu(A \cup B) = \mu(B) + \mu(A \backslash B)\text{.}\)
    🔗
    🔗

    Proof.

    \(A \cup B = B \cup (A \backslash B)\) is a disjoint union. The formula follows by finite additivity.
    🔗
    🔗

    Theorem 5.6.

    There is no set function \(\lambda: P(\R) \rightarrow [0, \infty]\) satisfying all of the following:
    1. Translation invariance.
      🔗
    2. Monotonicity.
      🔗
    3. Normalization.
      🔗
    4. Countable additivity. If \(\{A_i\}_{i = 1}^\infty\) is pairwise disjoint, then \(\lambda\left(\bigcup_{i = 1}^\infty A_i\right) = \sum_{i = 1}^{\infty} \lambda(A_i)\text{.}\)
      🔗
    🔗
    🔗

    Example 5.7. Vitali set.

    Given \(x, y \in \R\text{,}\) say \(x \sim y\) if \(x - y \in \Q\text{.}\) \(\sim\) defines an equivalence relation on the real line and let \(E_x = \{y \sim x\} = x + \Q\text{.}\) By the Axiom of Choice, for each equivalence class, pick a representative \(z_\alpha \in [0, 1]\text{.}\)
    🔗
    Then, \(B = \{z_\alpha\} \subseteq [0, 1]\) is a Vitali set. Given \(q \in \Q\text{,}\) let \(B_q = B + q\text{.}\)
    🔗
    🔗

    Observation 5.8.

    1. If \(y \in \R\text{,}\) \(y = z_\alpha + q\) for some \(z_\alpha\text{,}\) so \(y \in B_q\text{.}\)
      🔗
    2. \(B_q \cap B_p = \emptyset\) if \(p \ne q \in \Q\text{.}\)
      🔗
    🔗

    Lemma 5.9.

    If \(\lambda\) exists, then \(\lambda(B) > 0\text{.}\)
    🔗
    🔗

    Proof.

    \(\R = \bigsqcup_{q \in \Q} B_q\text{.}\) Thus, \(\mu(\R) = \sum_{q \in \Q} \lambda(B_q) = \sum_{q \in \Q} \lambda(B)\text{.}\) Since \(\lambda(\R) \ge \lambda([0, 1])\text{,}\) \(\lambda(B)\) cannot be \(0\) so \(\lambda(B) > 0\text{.}\)
    🔗
    🔗

    Lemma 5.10.

    If \(\lambda\) exists, then \(\lambda([0, 2]) = \infty\text{.}\)
    🔗
    🔗

    Proof.

    \(\{q_i\} = \Q \cap [0, 1] = Q\text{.}\) \(\bigcup_{i = 1}^\infty B_{q_i} \subseteq [0, 2]\text{.}\) But, \(\lambda\left(\bigcup_{i = 1}^\infty B_{q_i}\right) = \sum_{i = 1}^{\infty} \lambda\left(B_{q_i}\right) = \infty\text{.}\)
    🔗
    On the other hand, by translation invariance, \(\lambda([0, 2]) \le \lambda([0, 1]) + \lambda([1, 2]) = \lambda([0, 1]) + \lambda([0, 1]) = 2\text{.}\)
    🔗
    🔗

    Claim 5.11.

    Let \(N = [0, 1] \backslash B\text{.}\)
    🔗
    Then, \(\mu^*(N) = 1\text{,}\) \([0, 1] = N \sqcup B\text{.}\) \(\mu([0, 1]) = 1 < \mu(N) + \mu(B) = 1 + \varepsilon\text{.}\)
